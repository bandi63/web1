JavaScript

JavaScript is a high-level, often just-in-time compiled, multi-paradigm language. It conforms to the ECMAScript specification,[14] and has dynamic typing, prototype-based object-orientation, and first-class functions. It supports event-driven, functional, and imperative programming styles. It has application programming interfaces for working with text, dates, regular expressions, standard data structures, and the Document Object Model, or DOM.
